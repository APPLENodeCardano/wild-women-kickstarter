# Built by Women, For Women, In the Wild — Kickstarter Pre-Launch Site

A pre-launch campaign page for a women's wilderness retreat center in Southern Oregon. This site is designed to build community, collect early interest, and support a Kickstarter crowdfunding campaign.

## About the Project

This retreat center is being built **by women, for women** — in the wild lands of Southern Oregon. The campaign invites 10,000 founding women to participate through attendance, sponsorship, hosting, or a $1 pledge.

### Three Inaugural Retreats

| Retreat | Theme | Audience |
|---------|-------|----------|
| **The Return** | Wellness · Healing · Restoration | Women who've given everything to everyone else |
| **The Voice** | Creative Arts · Expression · Voice | Women ready to make something that matters |
| **The Crossing** | Transition · Leadership · Sisterhood | Women standing at life's most significant doorways |

## Tech Stack

- Pure HTML5, CSS3, and Vanilla JavaScript
- Google Fonts: DM Serif Display, Jost, Cormorant
- No build tools required — open `index.html` directly in any browser

## Getting Started

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/wild-women-kickstarter.git

# Navigate into the project
cd wild-women-kickstarter

# Open in browser
open index.html
# or serve locally
npx serve .
```
